### Essential Patterns
```css
/* Centering with Flexbox */
.container {
    display: flex;
    justify-content: center;
    align-items: center;
}

/* Responsive Grid */
.grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 1rem;
}

/* Mobile-first Media Query */
.element {
    width: 100%;
}

@media (min-width: 768px) {
    .element {
        width: 50%;
    }
}
```
