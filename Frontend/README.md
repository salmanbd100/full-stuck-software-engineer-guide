## üéØ Study Plan

### Beginner Track (8-10 weeks)
**Weeks 1-2: JavaScript Fundamentals**
- Data types, functions, scope
- Closures, this keyword
- Promises and async/await

**Weeks 3-4: HTML & CSS**
- Semantic HTML, accessibility
- Flexbox and Grid
- Responsive design

**Weeks 5-7: React Basics**
- Components and props
- Hooks (useState, useEffect)
- Basic state management

**Weeks 8-10: Testing & Performance Basics**
- Basic unit testing
- Performance fundamentals
- Portfolio projects

### Intermediate Track (10-12 weeks)
**Weeks 1-3: JavaScript Deep Dive**
- All JavaScript topics
- Event loop, prototypes
- Advanced ES6+ features

**Weeks 4-5: Advanced CSS**
- CSS animations, preprocessors
- CSS-in-JS solutions
- Advanced layouts

**Weeks 6-9: React Advanced**
- Advanced hooks, custom hooks
- Context API, state management
- Performance optimization
- React Router

**Weeks 10-11: TypeScript**
- Types and interfaces
- Generics and utility types
- React with TypeScript

**Week 12: Testing**
- Unit and integration tests
- E2E testing basics

### Advanced Track (12-16 weeks)
**All intermediate topics plus:**

**Weeks 13-14: Next.js**
- SSR/SSG/ISR strategies
- App Router
- Production optimization

**Weeks 15-16: Performance & Testing**
- Core Web Vitals optimization
- Advanced testing strategies
- CI/CD integration

## üíº Interview Preparation Checklist

### Before the Interview
- [ ] Review company's tech stack
- [ ] Practice coding problems on your strongest framework
- [ ] Prepare 3-5 projects to discuss in detail
- [ ] Review your portfolio: [salmanrahman.com](https://salmanrahman.com)
- [ ] Prepare questions about team, tech stack, and projects

### Technical Skills to Master
**JavaScript (Must-Have)**
- [ ] Closures and scope
- [ ] Promises and async/await
- [ ] Event loop understanding
- [ ] ES6+ features (destructuring, spread, modules)
- [ ] Array methods (map, filter, reduce)

**React (Essential for most roles)**
- [ ] Hooks (useState, useEffect, useContext, custom hooks)
- [ ] Component lifecycle
- [ ] State management patterns
- [ ] Performance optimization

**HTML/CSS (Foundation)**
- [ ] Semantic HTML and accessibility
- [ ] Flexbox and Grid
- [ ] Responsive design principles
- [ ] CSS animations

**TypeScript (Increasingly required)**
- [ ] Basic types and interfaces
- [ ] Generics
- [ ] React with TypeScript

**Performance (Senior roles)**
- [ ] Core Web Vitals
- [ ] Lazy loading and code splitting
- [ ] Bundle optimization

**Testing (Mid to Senior)**
- [ ] Unit testing with Jest
- [ ] Component testing with RTL
- [ ] E2E testing concepts

### Behavioral Preparation
- [ ] Prepare STAR method examples
- [ ] Review past projects and challenges
- [ ] Practice explaining technical decisions
- [ ] Prepare questions about company culture

## üè¢ Company-Specific Preparation

**Focus Areas:**
- Strong JavaScript fundamentals
- Algorithm knowledge (see DSA guide)

**Focus Areas:**

- Practical problem-solving
- Project experience
- Modern framework expertise
- Deployment and DevOps basics

**Common Interview Formats:**
- Take-home assignments
- Pair programming
- Architecture discussions
- Portfolio review
